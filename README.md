# CodeSync

**Real-Time Collaborative Code Editor**

CodeSync is a modern, real-time collaborative code editor that allows multiple users to write and execute code together in shared rooms. Built with React, Node.js, Socket.io, and Three.js.

---

## Features

- Real-time code collaboration
- 15+ programming languages supported
- Live code execution via Piston API
- Beautiful 3D animated backgrounds with Three.js
- Glassmorphism UI with aurora theme
- No signup required — just create a room and share
- Zero database — lightweight and fast

---

## Tech Stack

**Frontend**
- React 18 + Vite
- Tailwind CSS
- Three.js + React Three Fiber
- Monaco Editor
- Socket.io Client
- Zustand
- Framer Motion

**Backend**
- Node.js + Express
- Socket.io
- In-memory storage
